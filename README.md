# dicey-invite

Static invite landing page for the [dicey](https://github.com/bozulonas/Clock-Chrome-Extension) Chrome extension.

When a user shares an invite link (e.g. `https://bozulonas.github.io/dicey-invite/join#RoomName`), this page displays the room name and lets the dicey extension's content script auto-join.

## Setup

This repo is deployed via **GitHub Pages** from the root (`/`) of the `main` branch.

Enable it in: **Settings > Pages > Source: Deploy from a branch > Branch: main / root**
